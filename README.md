# Jogo-da-Cobrinha-com-pygame-
Jogo clássico da cobrinha desenvolvido em Python usando a biblioteca Pygame.
---

## 📦 Requisitos

Antes de começar, você precisa ter instalado:

* Python 3.x
* pip (gerenciador de pacotes do Python)

---

## 🔧 Instalação

### 1. Clone ou baixe o projeto

Se estiver usando Git:

```bash
git clone https://github.com/seu-usuario/jogo-cobrinha.git
cd jogo-cobrinha
```

Ou simplesmente baixe os arquivos e extraia em uma pasta.

---

### 2. Instale o Pygame

No terminal ou prompt de comando, execute:

```bash
pip install pygame
```

---

## ▶️ Como rodar o jogo

Dentro da pasta do projeto, execute:

```bash
python nome_do_arquivo.py
```

> Substitua `nome_do_arquivo.py` pelo nome do seu arquivo (ex: `cobra.py`)

---

## 🎮 Controles

* ⬅️ Seta para esquerda: mover para esquerda
* ➡️ Seta para direita: mover para direita
* ⬆️ Seta para cima: mover para cima
* ⬇️ Seta para baixo: mover para baixo

---

## 🧠 Como jogar

* Controle a cobrinha e coma a comida (quadrado vermelho)
* Cada comida aumenta o tamanho da cobra
* Evite bater nas paredes ou no próprio corpo
* O jogo termina quando ocorrer uma colisão
